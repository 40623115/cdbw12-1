gm用戶登入設置

操作步驟:

1.先git clone cd2020pj1倉儲。

2.建立新專案。

3.點選啟用API和服務，點選Google+ Domains API並點選管理。

4.點選憑證及設定同意畫面。

5.點選憑證建立OAuth用戶端ID。

6.設定來源及URL

7.將ID及密碼複製到編輯器，存儲在tmp下。

8.開啟leo，並打開cd2020pj1.leo。

9.將路徑及名稱改為自己的。

10.存儲，並點選ROC。

11.輸入pip install authomatic。

12.進入pj1，打開wsgi.py，即完成。
